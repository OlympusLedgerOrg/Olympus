# Olympus Copilot Guide

- Olympus is an append-only, publicly verifiable ledger; avoid mutation and favor explicit provenance.
- Use BLAKE3 helpers from `protocol.hashes` for hashing and `canonical_json_bytes` for deterministic JSON.
- Keep FastAPI endpoints explicit about authentication/rate limits and prefer read-only patterns for auditors.
- Do not add new runtime dependencies unless required; reuse existing utilities in `protocol` and `api/schemas`.
