# Copilot Hints for API code

- Prefer `api.schemas` models for request/response typing and keep endpoints read-only unless explicitly authenticated.
- Apply `RateLimit` and `RequireAPIKey` for FOIA write operations; public protocol endpoints stay open but validated.
- Reuse helpers in `api.services.storage_layer` and `protocol` rather than duplicating logic.
- Return canonical, verifier-friendly responses that include hashes and signatures when available.
